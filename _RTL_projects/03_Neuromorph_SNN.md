## Project Overview

Implementation on VC-707 FPGA :
- **HDL coding** : SystemVerilog
- **LUT resource**: 3488
- **FF count**: 3029
- **Static Power Consumption**: 108 mW
- **Dynamic Power Consumption**: 216 mw
- **Maximum Operating Frequency**: 118 MHz
